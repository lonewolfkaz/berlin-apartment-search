# Berlin Apartment Search

Personal apartment buying triage tool for evaluating Berlin listings against budget, district, school, and transit criteria.

![Screenshot](screenshot.png)

## Tech Stack

- React 19
- Vite 6
- Tailwind CSS v4
- React Router v7
- localStorage persistence (no backend)

## Quick Start

```bash
npm install
npm run dev
```

Opens at [http://localhost:5173](http://localhost:5173).
